What is Loops?

  • Loops is mainly used for iterating over arrays or objects if you have to run the same code several times. 

                const items = ["Item 1", "Item 2", "Item 3", "Item 4", "Item 5", "Item 6", "Item 7"];
                //to print the items always is running items[0] -> just the index is changing
                console.log(items[0]);  // index: 0
                console.log(items[1]);  // index: 1
                console.log(items[2]);  // index: 2
                console.log(items[3]);  // index: 3
                console.log(items[4]);  // index: 4
                console.log(items[5]);  // index: 5
                console.log(items[6]);  // index: 6

Solution


                const items = ["Item 1", "Item 2", "Item 3", "Item 4", "Item 5", "Item 6", "Item 7"];
                for (let i = 0; i < items.length; i++) {
                    console.log(items[i]);
                }
