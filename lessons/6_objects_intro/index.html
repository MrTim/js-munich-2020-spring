Objects in a nutshell

  • Objects consist of keys and values
  • The key is like the name of the variable
  • The value is like a variable itself


              let book = {
                title: "Raising goats for dummies",
                image: "goats.jpg",
                pageCount: 340,
                price: 15
              };

What can we do with objects?

Get value of it's attributes


              let book = {
                title: "Raising goats for dummies",
                image: "goats.jpg",
                pageCount: 340,
                price: 15
              };

          alert("Book name is " + book.title);

Change value of it's attributes


              let book = {
                title: "Raising goats for dummies",
                image: "goats.jpg",
                pageCount: 340,
                price: 15
              };

              book.pageCount = 900;

              alert(book.title + " has " + book.pageCount + " pages");

And even add new attributes


              let book = {
                title: "Raising goats for dummies",
                image: "goats.jpg",
                pageCount: 340,
                price: 15
              };

              book.rating = "5 stars";

              alert(book.title + " book was rated with " + book.rating);

              let person = {

              alert(person.name + "'s favorite book is " + person.favoriteBook.name);
